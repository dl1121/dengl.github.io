CSS弹性布局 Flex
Created2020-07-29|Updated2020-07-30|CSS
|Post View:

弹性布局是新一代的布局方式，在传统的布局中使用浮动 float
而传统的浮动布局致命的缺点就是浮动元素一旦脱离了文档流，就无法撑起父元素，会造成父元素的高度塌陷。
CSS的代码将高度依赖于HTML代码结构等等….

使用Flex布局

要使用弹性布局，需要在父元素上设置 display:flex;属性

注意： 给元素设置了Flex布局以后，它的子元素的float、clear、和vertical-align三个属性将会失效。

行内元素使用弹性布局，需要在父元素设置 display:inline-flex 属性

特点：

弹性布局中开启弹性布局的元素成为 容器 把弹性布局中的综援素称为项目

弹性布局的容器的两根轴(主轴和交叉轴)

主轴：所排列的方向为主轴(若横向排列则主轴为X轴，交叉轴为Y轴)

交叉轴：与主轴相交叉的周

在弹性布局中项目的尺寸，默认情况下(不设置宽高)，占据主轴方向的尺寸内容大小，交叉轴的尺寸尽量的占满整个容器

属性值：

flex-direction:;设置主轴方向（设置布局方向）

值 说明
row 默认值，从左至右水平排列
row-reverse 从右至左水平排列
column 从上至下垂直排列
column-reverse 从下至上垂直排列

justify-content:;设置子项目在主轴方向的对齐方式

值 说明
center 居中（根据主轴方向居中）
flex-start 对齐开始的位置
flex-end 对齐结束的位置
space-between 两端对齐,项目之间的间隔相等，项目于边框之间没有距离
space-around 分散对齐，项目之间的间隔相等，项目与边框之间的距离是项目之间据距离的一半

align-items:;设置子项目在交叉轴的对齐方式

值 说明
center 居中（根据交叉轴方向居中）
flex-start 对齐开始的位置
flex-end 对齐结束的位置
stretch 默认值 如果项目未设置高度或者是auto将沾满整个容器的高度
baseline 基线对齐

align-self:;单独设置子项目交叉轴的对齐方式 设置项目自身在交叉轴上的对齐方式，会覆盖父标签的align- items的值。

值 说明
center 居中
flex-start 对齐开始的位置
flex-end 对齐结束的位置
stretch 默认值 如果项目未设置高度或者是auto将沾满整个容器的高度
baseline 基线对齐
auto 默认取父标签align-items的值
