CSS3过渡
Created2020-07-31|Updated2020-08-01|CSS
|Post View:

过渡：让我们在限定的时间内 从一个属性值到另一个属性值的转变…
而在没有过渡属性的时候，当一个元素的属性值发生变化时，
浏览器就会将个这个元素瞬间渲染成新属性值的样式…
为了产生视觉上的变化效果，通常会使用到过渡属性…

过渡(transition)：

1. 属性transition-property：想要设置过渡的属性名；（比如：transform, width, height, backgroud, color, ）

3. transition-duration: 过渡的时间长度；

4. transition-timing-function: 过渡方式（linear, ease, ease-in, ease-out, ease-in-out）；

5. transition-delay: 延迟时间；

简写：

transition： 想要设置过渡的属性名 过渡的时间长度 过渡方式 延迟时间

例：transition:width 3s linear 0; 表示该元素的宽度会在三秒内匀速的变化

补充：

transition-property：规定设置过渡效果的css属性名称。如果没有设置属性名称，所有属性都会获得的过渡效果（all）

transition-timing-function: 过渡方式

值 说明
linear 匀速
ease 先加速后减速
ease-in 加速
ease-out 减速
ease-in-out 等同于easy

transition 属性仍有一定的局限性，他的触发事件基本需要结合滑动，点击，焦点事件时使用，无法在网页加载的时候自动发生，与JavaScript结合使用时无效

transform（转换动画、直接动画）

属性值

属性 说明 值
rotate 旋转 rotatex、rotatey、rotatez（默认
translate 平移 translatex（默认）、translatey、translatez
skew 变形 skewx（默认）、skewy
scale 伸缩 两个参数时，分别是宽度和高度的比例;一个参数时，为等比例缩放
